### Open Source Dependencies

This project is built on the shoulders of giants. Special thanks to the maintainers and contributors of:

#### Frontend
- **Next.js** - React framework
- **React** - UI library
- **React Hot Toast** - Notification system

#### Backend
- **Go** - Application runtime
- **Node.js** - Service runtime
- **WebView** - Native window integration

#### Build Tools
- **NSIS** - Windows installer
- **Puppeteer** - Browser automation
- **pkg** - Node.js executable compiler
- **standard** - JavaScript linting

#### Data & APIs
- **axios** - HTTP client
- **nedb-promises** - Embedded database
- **ws** - WebSocket implementation
- **glob** - File pattern matching
